NEORETRO

This piece is a responsive web app I designed for a course project where the brief was to create an online store. I created an e-commerce concept offering quality motorcycle goods with free shipping and returns worldwide. I built this project from start to finish as the UI/UX designer. I started by creating the company name, branding and logo designs. Based on the project brief, I created user personas and user flows to begin to map out the user’s experience. Low fidelity wire framing lead to digitized mid fidelity designs which were tested by users. This entire process helped to create an intuitive and beautiful user interface in the final design.

My Process

User First

Using the stories provided by the project brief, the users were conceptualized and their goals were laid out.

Ideation

I drew inspiration from several other e-commerce sites that have clean designs: most notably Prana, Nordstrom and P&Co. These sites helped shape my user interface and the wireframe designs.

Visual Design

I wanted the logo design to be elegant but also coincide with the modern classic styling of the products being sold. I decided to combine a modern typeface with a retro inspired typeface and let that drive the branding. I also created a secondary logo design that is more condensed so it can be used when space is limited and the full logo could be compromised. The UI elements were designed to communicate quality and elegance and the color palette to do the same with muted, vintage inspired colors.

Mockups

I then took the finished designs and created mockups to showcase the beautiful, responsive user interface.
